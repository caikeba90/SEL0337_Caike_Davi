# SEL0337_Caike_Davi

A etapa 1 do projeto foca na configuração do Init System da Raspberry Pi utilizando o systemd para automatizar a execução de aplicações embarcadas. A implementação consistiu na criação de uma unidade de serviço personalizada  capaz de gerenciar um script em Python para controle de GPIO, definindo explicitamente as rotinas de execução (ExecStart) e de encerramento do processo. Através do utilitário systemctl, o serviço foi habilitado para iniciar automaticamente durante o boot da raspberyy, garantindo que a aplicação opere em segundo plano imediatamente após o carregamento do sistema operacional, sem a necessidade de login ou intervenção manual, dessa forma foi criado um código em python que foi inserido na rotina de execução para que ao ligar um led fica piscando e ao solicar o encerramento do processo por meio do "ExecStop" o led fica ligado continuamente.
